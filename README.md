# Ilin.PartnerApp

Настольное приложение на **WPF + .NET + Entity Framework Core + PostgreSQL** для учета партнеров, товаров и продаж.

## Возможности

- ведение списка партнеров;
- учет типов партнеров;
- управление товарами;
- оформление и просмотр продаж;
- автоматический расчет скидки партнера;
- сохранение исторической цены и скидки на момент продажи;
- модульные тесты на **MSTest**.

## Стек технологий

- .NET 8
- WPF
- Entity Framework Core
- PostgreSQL
- MSTest

## Структура проекта

- `Ilin.PartnerApp.Wpf` — пользовательский интерфейс
- `Ilin.PartnerApp.Lib` — сущности, контекст БД, сервисы, бизнес-логика
- `Ilin.Partner.Tests` — модульные тесты

## Настройка базы данных

Перед запуском приложения необходимо создать базу данных, схему и таблицы.

### 1. Создание роли и базы данных

Выполнить от пользователя `postgres`:

```bash
psql -U postgres -d postgres -f 01_create_db_ilin.sql
```
### 2. Создание схемы и таблиц

Выполнить в базе ilin:
```bash
psql -U postgres -d ilin -f 02_create_schema_and_tables.sql
```

### 3. Заполнение тестовыми данными

Выполнить в базе ilin:

```bash
psql -U postgres -d ilin -f 03_seed_test_data.sql
```

### Параметры подключения

Приложение использует следующие параметры подключения к PostgreSQL:

- Host: `localhost`

- Port: `5432`

- Database: `ilin`

- Username: `app`

- Password: `123456789`

Строка подключения:
```bash
Host=localhost;Port=5432;Database=ilin;Username=app;Password=123456789
```

### Тестовые данные

Скрипт `03_seed_test_data.sql` добавляет:


- 3 типа партнеров

- 6 партнеров

- 10 товаров

- 17 тестовые продажи

## Основные бизнес-правила

- скидка рассчитывается по объему продаж партнера;

- неактивные товары нельзя использовать в новых продажах;

- при продаже сохраняются:

    - базовая цена товара,

    - скидка,

    - цена продажи на момент оформления;

- нельзя удалить партнера, если у него есть продажи;

- нельзя удалить товар, если он используется в продажах;

- нельзя удалить тип партнера, если он используется в карточках партнеров.
